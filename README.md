## Remarques / Commentaires / Motivations

Le problème 1 a été choisi car il permet de mettre en œuvre plusieurs notions clés du cours GNU/Linux avancé, notamment :

- l’automatisation de tâches via `cron` ;
- l’écriture de scripts shell robustes et relançables ;
- le traitement de données via `awk` ;

La solution proposée repose sur un processus automatisé d’archivage et d’anonymisation des données personnelles, ainsi que sur la génération annuelle de rapports d’activité.  
Les scripts fournis sont indépendants de l’utilisateur et du chemin d’installation, afin de garantir la portabilité et la reproductibilité de la solution après clonage du dépôt.
